## Get Started

- Add the repo
```bash
helm repo add m4tt4r https://ahmedmattar21.github.io/charts/ 
```
```bash
helm repo update
```

- Search for your app
```bash
helm search repo memo
```

- Install the app
```bash
helm install memo m4tt4r/memo
```
